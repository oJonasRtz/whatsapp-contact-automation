# whatsapp-contact-automation

A Python application that retrieves contacts from Supabase and sends personalized WhatsApp messages through Z-API.

## How to Run 

### Using Makefile
- Run ```make``` on your terminal. It will install all dependencies and run the program   

### Manual Setup
- ```pip install -r requirements.txt```
- ```python3 main.py```   

## Env
- Create a `.env` file with the following variables:
```txt
SUPABASE_URL=""
SUPABASE_KEY=""
ZAPI_INSTANCE_ID=""
ZAPI_TOKEN=""
```

## Database schema used
- The program expects a `contacts` table following this schema
```sql
create table contacts (
    id bigint generated by default as identity primary key,
    name text not null,
    phone text not null
);
```
